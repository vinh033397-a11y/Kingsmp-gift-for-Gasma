KING SMP - BỘ QUÀ 16 NGÀY

Trang web đã được tạo từ 16 trang Day 1 đến Day 16 của file Word.
Trang giới thiệu đầu tiên đã được bỏ qua.

NGÀY MỞ KHÓA HIỆN TẠI
- Day 1: 25/07/2026 lúc 00:00 (giờ Việt Nam)
- Mỗi Day tiếp theo tự mở sau 1 ngày.

ĐỔI NGÀY MỞ DAY 1
1. Mở file config.js bằng Notepad.
2. Tìm dòng:
   firstUnlockDate: "2026-07-25T00:00:00+07:00",
3. Thay ngày và giờ nhưng giữ nguyên định dạng.

VÍ DỤ
- 28/07/2026 lúc 00:00:
  firstUnlockDate: "2026-07-28T00:00:00+07:00",

CÁCH ĐĂNG LÊN GITHUB PAGES
1. Tạo repository mới trên GitHub, nên đặt tên king-smp-gifts.
2. Giải nén file ZIP này.
3. Upload TẤT CẢ nội dung bên trong thư mục lên repository.
   File index.html phải nằm ở ngoài cùng, không nằm trong thư mục con.
4. Vào Settings > Pages.
5. Source: Deploy from a branch.
6. Branch: main; Folder: / (root); sau đó Save.
7. Chờ GitHub tạo đường link website.

LƯU Ý
GitHub Pages là website tĩnh. Cơ chế khóa này rất phù hợp để giữ bất ngờ cho người dùng bình thường,
nhưng không phải khóa bảo mật tuyệt đối: người biết kỹ thuật vẫn có thể tìm ảnh trong repository công khai.
